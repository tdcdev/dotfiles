Linux dotfiles
==============

Installation
------------

Install following requirements:

    zsh                 \
    subversion          \
    mercurial           \
    git                 \
    gitk                \
    git-gui             \
    vim                 \
    vim-gtk             \
    make                \
    cmake               \
    curl                \
    libpython2.7-dev    \

Then run the following commands:

    bash setup.sh
    bash build_llvm.sh
    bash build_ycm.sh
